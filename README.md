# 💊 MedRemind - App Design (Figma)

MedRemind is a medicine reminder app designed to help patients and caregivers manage medication schedules effectively.

This project focuses on user experience design and workflow planning through a structured wireframe.

## 🎯 Objective

To create an intuitive and user-friendly system for medication tracking, reminders, and caregiver collaboration.

## ✨ Key Features (Wireframe)

* 🔐 Register & Login
* 🏠 Home Dashboard
* 💊 Medicine Schedule Management
* 📊 Medication Reports
* 🤝 Patient–Caregiver Referral Connection
* 💳 Premium feature system (Ad-free & advanced reports)

* Clear navigation flow
